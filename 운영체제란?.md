# 운영체제 (Operationg System) :computer:
  - 자원, 하드웨어(HW) 들을 효율적으로 관리하여 유저들에게 서비스를 제공하는 것
  
## 컴퓨터 하드웨어
  - 프로세서 (Processor)
    - CPU
    - 그래픽카드 (GPU)
    - 응용 전용 처리장치 등..
 
  - 메모리 (Memory) : 무언가를 저장하는 장치
    - 주 기억장치 : DRAM
    - 보조 기억장치 : HDD, SSD
    
  - 주변장치
    - 키보드/마우스
    - 모니터 등..
    
### 프로세서
  - 컴퓨터의 두뇌 (CPU)
    - 
    - 연산 수행
    - 컴퓨터의 모든 장치의 동작을 제어 한다.
